# Introduction

**Mojuru** is a **bring-your-own-content client** for iOS.  
It allows you to watch and stream your own content by importing plugins, while also offering a personalized experience as an AniList client.

## Key Features

### Plugin Support

Mojuru enables you to **add your own plugins** to access and stream content.  
> For detailed instructions on adding plugins, see [Adding Plugins Guide](/docs/guides/getting-started#adding-plugins).

### AniList Integration

Mojuru acts as a **full-featured AniList client**, letting you:

- Search for series and movies
- Track your watching/reading progress
- Manage your personal library
> For login instructions, see [AniList Guide](/docs/guides/anilist#logging-in-with-anilist).

### Library Management

### Personalization

Mojuru allows for **customization of the user experience**, including:

- Theme changes  
- App icon changes  
- Additional interface tweaks  

Some personalization features are **unlocked with Mojuru Plus**.  
> Learn more about [Mojuru Plus](/docs/mojuru-plus).

## Mojuru Plus

**Mojuru Plus** is the special edition of the app, unlockable via donation.  
It provides perks such as:

- Theme customization  
- App icon customization  
- Picture-in-picture mode in the video player  
- Other exclusive tweaks  

> See the [Mojuru Plus Guide](/docs/guides/mojuru-plus) for instructions on unlocking Mojuru Plus.
