# Download

**Mojuru** is an iOS-only application.  
It requires **iOS 15.1 or later**.

> Android support is **not available at the moment**, but it may be considered in the future.

Currently, there are **two supported ways** to install Mojuru on your device.

## Install via IPA

You can install Mojuru manually using its `.ipa` file.

### Manual download

The `.ipa` file is available from the official [**GitHub repository**](https://www.mojuru.app/go/mojuru-repo-releases).

1. Download the latest `.ipa` release from GitHub.
2. Install it using a compatible sideloading method of your choice.

### Using AltStore (Classic)

To simplify the installation process, Mojuru provides an **AltStore Classic source**.

Requirements:
- **AltStore Classic** must be installed and configured on your device  
  → Source URL: [classic source](https://aleganza-altstore.vercel.app/classic-source.json)

Steps:
1. Set up AltStore Classic on your iPhone or iPad.
2. Add the Mojuru source to AltStore.
3. Install Mojuru directly from AltStore.

## Install via AltStore PAL

If you are located in the **European Union**, Mojuru can also be installed using **AltStore PAL**.

Requirements:
- **AltStore PAL** must be available in your region  
  → Source URL: [PAL source](https://aleganza-altstore.vercel.app/pal-source.json)
- AltStore PAL must be properly installed and configured on your device.

Steps:
1. Install and configure AltStore PAL.
2. Add the Mojuru **AltStore PAL source**.
3. Download and install Mojuru directly from AltStore PAL.

## Unsupported distribution methods

The following distribution methods are **not supported**:

- **TestFlight**
  - There is no TestFlight release planned at this time.
